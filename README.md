# My recipe journal

Application mobile développé dans le cadre d'un projet à l'ESILV.

# But de l'application

Application simple de recherche de recette de cuisine grâce à l'API Edamane.
Elle permet de rechercher et filter des recherches en fonction de certains critères (sans-sucre, pauvre en gras, sans alcool, haut en protein,
végétarien, vegan....).
Vous pouvez également sauvegarder vos recettes favorites!
